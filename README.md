# FinFlow - Financial Management Platform

A comprehensive full-stack financial management platform built with C# .NET and React.

## Tech Stack

### Backend
- C# / .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Stripe API

### Frontend (Coming Soon)
- React
- Next.js
- TypeScript
- TailwindCSS

## Setup Instructions

### Prerequisites
- .NET 8 SDK
- PostgreSQL 16+
- VS Code with C# Dev Kit

### Getting Started

1. Clone the repository
```bash
git clone <your-repo-url>
cd FinFlow
```

2. Restore packages
```bash
dotnet restore
```

3. Build the solution
```bash
dotnet build
```

4. Run the API
```bash
cd src/FinFlow.Api
dotnet run
```
